## ✨ 새로운 디자인 하이라이트

### 1. 그라데이션 헤더 🌈
**AS-IS:**
- 단순한 흰색 배경
- 평범한 제목

**TO-BE:**
- **주황색 그라데이션 배경** (#FF6B35 → #F7931E)
- 화이트 텍스트로 강렬한 대비
- 원형 장식 요소로 입체감
- 로고 + 회사명 좌측 배치
- "견적서" 우측에 큰 글씨로 강조

### 2. 메타 정보 바 📊
- 반투명 카드 디자인
- 견적번호, 발행일, 건명을 한눈에
- 블러 효과로 세련된 느낌

### 3. 고객/공급자 정보 카드 💳
**Customer 카드:**
- 화이트 배경
- 주황색 아이콘 (👤)
- 호버 시 살짝 올라오는 효과
- 그림자로 입체감

**Supplier 카드:**
- **다크 그라데이션 배경** (#2c3e50 → #34495e)
- 화이트 텍스트
- 프리미엄한 느낌
- 아이콘 (🏢)

### 4. 견적 테이블 개선 📋
**헤더:**
- 다크 그라데이션 배경
- 대문자 영문 레이블
- 깔끔한 정렬

**내용:**
- 호버 시 살짝 배경색 변경
- 품목명 진하게 강조
- 설명문구 회색으로 구분
- **이미지 썸네일 표시** (60×60px)
- 클릭 시 이미지 확대 팝업

### 5. 가격 요약 박스 💰
- 회색 배경 카드
- 최종 합계는 주황색 그라데이션 텍스트
- 큰 폰트로 강조 (32px)
- 상단 3px 검은 테두리

### 6. 안내사항 박스 📌
**고정 메시지:**
- 파란색 배경 (#e3f2fd)
- 파란 좌측 테두리 (5px)
- 📌 아이콘

**특이사항:**
- 노란색 배경 (#fff8e1)
- 주황 좌측 테두리
- 📝 아이콘

### 7. 푸터 🏢
- 회색 배경
- 주황색 상단 테두리
- 회사명 큰 글씨
- 주소
- 감사 메시지 (이탤릭)
- 직인 우측 하단에 표시

### 8. 액션 버튼 🔘
- 우측 상단 고정
- 화이트 버튼
- 호버 시 주황색 변경
- 그림자 효과
- "📷 이미지 저장" 버튼

## 🚀 주요 기능

### 1. 이미지 확대 보기
- 품목 이미지 클릭
- 전체 화면 모달로 크게 보기
- 배경 클릭으로 닫기

### 2. 이미지 저장
- 우측 상단 버튼
- html2canvas로 고해상도 PNG 저장
- 파일명: 견적서_고객명_날짜시간.png

### 3. 인쇄 최적화
- 인쇄 시 액션 버튼 숨김
- A4 용지에 최적화
- 배경 제거
- 페이지 나눔 방지

### 4. 반응형 효과
- 카드 호버 시 올라감
- 테이블 행 호버 시 배경색
- 버튼 호버 시 색상 변경
- 이미지 호버 시 확대

## 📦 파일 정보

**파일명:** quote_view_v5_프리미엄디자인.php

**사용 방법:**
1. 기존 quote_view.php 백업
   ```bash
   cp quote_view.php quote_view_backup.php
   ```

2. 새 파일로 교체
   ```bash
   cp quote_view_v5_프리미엄디자인.php quote_view.php
   ```

3. 권한 설정
   ```bash
   chmod 755 quote_view.php
   ```

4. 브라우저 캐시 클리어
   - Ctrl + F5
